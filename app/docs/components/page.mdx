# Components Overview

Documentation Hub comes with a comprehensive set of pre-built components that you can use in your MDX files to create rich, interactive documentation.

## Available Components

### Layout Components

These components help structure your content:

- **Card** - Container for grouping related content
- **Alert** - Highlight important information
- **Badge** - Label and categorize content
- **Button** - Interactive elements

### Typography Components

All standard HTML elements are styled and enhanced:

- Headings (h1-h6)
- Paragraphs
- Lists (ordered and unordered)
- Blockquotes
- Code blocks and inline code

## Component Examples

### Cards

Cards are perfect for organizing content into digestible sections:

<Card title="Basic Card" description="A simple card with title and description">
  This is the content area where you can include any MDX content, including
  other components, code blocks, and formatted text.
</Card>

<Card title="Advanced Features">
  Cards can contain complex content: - Lists and bullet points - Code examples -
  Other nested components - **Bold** and *italic* text
</Card>

```mdx
<Card title="Basic Card" description="A simple card with title and description">

```

### Alerts

Use alerts to draw attention to important information:

<Alert type="info">
  **Info Alert**: Use this for general information and helpful tips.
</Alert>

<Alert type="warning">
  **Warning Alert**: Use this to highlight potential issues or important
  considerations.
</Alert>

<Alert type="success">
  **Success Alert**: Use this to confirm successful actions or highlight
  positive outcomes.
</Alert>

```mdx
<Alert type="info">Your informational message here.</Alert>

<Alert type="warning">Your warning message here.</Alert>

<Alert type="success">Your success message here.</Alert>
```

## Code Blocks

Syntax highlighting is built-in for many languages:

### JavaScript/TypeScript

```javascript
// React component example
import React, { useState } from "react";

function Counter({ initialValue = 0 }) {
  const [count, setCount] = useState(initialValue);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}

export default Counter;
```
