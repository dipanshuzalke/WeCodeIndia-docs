# Installation

Get started with Documentation Hub by installing the necessary packages and setting up your development environment.

## Prerequisites

Before you begin, make sure you have the following installed on your system:

- **Node.js** (version 16.0 or higher)
- **npm** or **yarn** package manager
- **Git** for version control

<Alert type="info">
  We recommend using the latest LTS version of Node.js for the best experience.
</Alert>

## Installation Methods

### Option 1: Using npm

```bash
npm install documentation-hub
```

### Option 2: Using yarn

```bash
yarn add documentation-hub
```

### Option 3: Using pnpm

```bash
pnpm add documentation-hub
```

## Quick Setup

After installation, create a new project:

```bash
npx create-docs-hub my-docs
cd my-docs
npm run dev
```

This will create a new Documentation Hub project with all the necessary files and dependencies.

## Project Structure

Your new project will have the following structure:

```
my-docs/
├── app/
│   ├── docs/
│   │   ├── layout.tsx
│   │   └── page.mdx
│   ├── globals.css
│   └── layout.tsx
├── components/
│   ├── ui/
│   ├── navbar.tsx
│   ├── sidebar.tsx
│   └── footer.tsx
├── lib/
│   └── utils.ts
├── package.json
└── next.config.js
```

## Configuration

### Basic Configuration

Create a `docs.config.js` file in your project root:

```javascript
module.exports = {
  title: 'My Documentation',
  description: 'Comprehensive documentation for my project',
  theme: {
    primary: '#3b82f6',
    dark: true
  },
  search: {
    enabled: true,
    provider: 'local'
  }
};
```

### Environment Variables

Create a `.env.local` file for environment-specific settings:

```bash
NEXT_PUBLIC_SITE_URL=http://localhost:3000
NEXT_PUBLIC_SITE_NAME=My Documentation
```

## Verification

To verify your installation is working correctly:

1. Start the development server:
   ```bash
   npm run dev
   ```

2. Open your browser to `http://localhost:3000`

3. You should see the documentation homepage

## Troubleshooting

### Common Issues

**Node.js version error**
```bash
error: Node.js version 16.0 or higher is required
```
Solution: Update Node.js to the latest LTS version.

**Port already in use**
```bash
Error: Port 3000 is already in use
```
Solution: Use a different port with `npm run dev -- -p 3001`

**Module not found**
```bash
Error: Cannot find module 'documentation-hub'
```
Solution: Ensure you've run `npm install` in your project directory.
