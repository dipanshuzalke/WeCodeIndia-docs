# Quick Start

Get up and running with Documentation Hub in just a few minutes. This guide will walk you through creating your first documentation page.

<Alert type="info">
  Make sure you've completed the [installation](/docs/installation) before following this guide.
</Alert>

## Create Your First Page

### Step 1: Create a New MDX File

Create a new file at `app/docs/my-first-page/page.mdx`:

```mdx
import { components } from '@/components/mdx-components';
export { components };

# My First Page

This is my first documentation page!

## Features

- Easy to write in Markdown
- Supports React components
- Beautiful syntax highlighting

```javascript
console.log('Hello, Documentation Hub!');
```

<Alert type="success">
  Your first page is ready!
</Alert>
```

### Step 2: Add Navigation

Update the navigation in `components/sidebar.tsx` to include your new page:

```typescript
const navigation = [
  {
    title: 'Getting Started',
    items: [
      { title: 'Introduction', href: '/docs' },
      { title: 'Installation', href: '/docs/installation' },
      { title: 'Quick Start', href: '/docs/quick-start' },
      { title: 'My First Page', href: '/docs/my-first-page' }, // Add this line
    ],
  },
  // ... rest of navigation
];
```

### Step 3: View Your Page

Start the development server and navigate to your new page:

```bash
npm run dev
```

Visit `http://localhost:3000/docs/my-first-page` to see your page in action!

## Code Highlighting

Documentation Hub automatically highlights code blocks:

### JavaScript

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

const message = greet('Documentation Hub');
console.log(message);
```

### Python

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Generate Fibonacci sequence
for i in range(10):
    print(fibonacci(i))
```

## Frontmatter

You can add metadata to your pages using frontmatter:

```mdx
---
title: My Custom Title
description: A custom description for SEO
author: John Doe
date: 2024-01-15
tags: [guide, tutorial, beginner]
---

# Page Content

Your page content goes here...
```
